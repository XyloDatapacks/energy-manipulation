==<TRANSLOCATION EFFECT SETTINGS>==

Max teleport distance.

Default value is -1

#xem.spell.run.effect.translocation.inter_dimensional_tp xem.op 
If a teleport can go between dimensions
1 is true, 0 is false
Default is 1
